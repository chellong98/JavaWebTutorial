Welcome on Futura
By Biswajit Saha • February 04, 2015 • 1 Comment
Futura is clean content focus theme for ghost blogging platform. It is responsive means it will fit in various device of diffrent size. Featuring an image or an audio or a video in your post is very simple as 1-2-3... Using Futura you can setup a beautiful personal blog. This theme is multi-user ready so you can setup a multi-user blog or journal with no extra effort.

Installing and customizing this theme is very easy. All required informations are provided in documentation.
Some feature highlight
  • Ready for Ghost-0.5.8
  • Widgetize Sidebar and Footer.
  • Add, Remove or Reorder Widgets.
  • supports image logo.
  • Multi author support.
  • Easily featured image / youtube / vimeo / soundcloud / mixcloud.
  • Beautiful Code Highlight.
  • static pages.
  • Two level of menu.
  • Cross-Browser Compatibility Chrome, FireFox, Safari, IE9, IE10, IE11
  • Comment with disqus
  • Google font
  • Font Awesome icons
  • And many more...
